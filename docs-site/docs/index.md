# Overview

Vottun AI Compliance (EU AI Act Art. 50) is an open-source toolkit for certifying, watermarking, detecting and verifying AI-generated content.

This repo contains:

- SDKs:
  - `sdk-python/`
  - `sdk-typescript/`
- MCP server tools:
  - `mcp-server/`
- Examples and integration templates:
  - `examples/`
