# Listing preparation

Directory listings often require:

1. A publishable npm package for the TypeScript SDK
2. A tool-card/capabilities description for the MCP server
3. Links to repo documentation (README)

Repo-owned prep steps are described here:

- `docs/sprint3_listing_prep.md`
- `docs/listing_tool_card.template.json`
- `listings/` (starter payload templates for Smithery/mcp.so/AgentHub/Composio/LangChain Hub/LlamaHub)
