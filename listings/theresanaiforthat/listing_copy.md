## Long description
Vottun AI Compliance helps teams certify, watermark, detect, and verify AI-generated content with an auditable workflow.  
It includes Python/TypeScript SDKs, an MCP server for agent tooling, and contract ABI integrations.  
Free testnet mode is available for quick trials; production usage scales via API key and enterprise options.

## Key features
- Certify generated content on-chain
- Verify by certificate id or content hash
- Detect watermark signals
- MCP tools: `certify_content`, `verify_certificate`, `detect_watermark`, `get_certificate`
